# Online Food Delivery Application

A full-stack food delivery application that connects **customers**, **restaurants**, and **delivery partners** through a single platform. The system supports restaurant onboarding, menu management, order placement, secure payments, real-time delivery tracking, and ratings & reviews.

---

## Project Overview

The Online Food Delivery Application acts as a bridge between restaurants and consumers. Customers can browse nearby restaurants, add food items to a cart, place orders, and track deliveries with dynamic ETA updates. Restaurants manage menus and orders via a dashboard, while delivery partners handle pickups and drop-offs with live location tracking.

---

## User Roles

* **Customer**: Sign up, browse restaurants, place orders, track deliveries, make payments, rate/review
* **Restaurant Owner**: Register restaurant, manage menu, accept orders, assign delivery partner
* **Delivery Partner**: Accept deliveries, update pickup/drop status, share live location
* **Admin**: Manage users, restaurants, orders, and view analytics
